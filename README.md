# Simple Chat

Simple chat with React, Node, <span>Socket.io</span>

## Usage
```
// in project's root directory
npm run setup
npm run start:client  // port 3000
npm run start:server  // port 4000
```

## Features

- [x] Set Nickname
- [x] Customize Nickname Color
- [x] Chat Each Other!
- [x] Show People List in Chat
